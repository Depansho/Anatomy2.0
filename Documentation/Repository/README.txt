How to get started:

1. Install XAMPP
2. Run XAMPP Control Panel
3. Start Apache and MySQL
4. Create Database as described in "Database.txt"
5. OPTIONAL: Configure virtual Hosts as described in "XAMPP for Windows-Creating Virtual Hosts.txt". This may be different on Linux.
6. Restart Apache
7. Open the webpage "3dmodels.local" in your web browser
